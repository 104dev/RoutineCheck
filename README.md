## 概要

カレンダーとTodoの良いとこどりを目指したアプリです。

## 特徴

- タスクの継続率をプロジェクト単位で管理
- タスクを連日、1週間毎、1ヶ月毎に一括生成が可能
- プロジェクトやタスクにはアクティビティという実行メモを残すことが可能
- 期限切れのタスクのステータスを自動的に「断念」に変更

## システム要件

- Swift 5.8
- iOS 16
- Xcode 14

## インストール

[App Store](https://apps.apple.com/jp/app/routine-checker/id6462190861) にてダウンロード
